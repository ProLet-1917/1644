# 使用说明
############################
# 1. 特权 (Privileges)
#    - 在 setup 模板中使用 privilege = { privilege_name } 格式
#    - 某些特权有潜在条件 (potential) 和允许条件 (allow)
#    - 某些特权互斥，不能同时使用
#
# 2. 法律 (Laws)
#    - 在 setup 模板中使用 laws = { law_name = policy_name } 格式
#    - 每个法律类型下有多个政策选项
#    - 某些法律需要特定政府类型或条件
#
# 3. 议会 (Parliament)
#    - 在 setup 模板中使用 parliament = { parliament_type = parliament_type:type_name } 格式
#    - 某些议会类型需要特定条件或修饰符
#
# 4. 国家类型 (Government Type)
#    - 在 setup 模板中使用 government = { type = government_type_name } 格式
#    - 国家类型决定可用的继承制和国家制度
#
# 5. 国家实体类型 (Country Entity Type)
#    - 在 setup 文件中使用 type = entity_type_name 格式（在国家定义块中）
#    - 用于特殊国家实体：army（军队国家）、building（建筑国家）、location（地点国家）、pop（人口国家）
#    - 这些不是政府类型，而是国家实体的分类
#
# 6. 继承制 (Heir Selection)
#    - 在 setup 模板中使用 heir_selection = heir_selection:type_name 格式
#    - 某些继承制需要特定政府类型、文化或条件
#
# 7. 国家制度 (Government Reforms)
#    - 在 setup 模板中使用 reforms = { reform_name } 格式
#    - 某些制度需要特定政府类型、时代或条件
#    - 某些制度互斥，不能同时使用
