# 征召兵数量 = estate人口 * levy_banner_cavalry_size * (1 + global_levy_size_modifier + estate_levy_size_modifier + local_levy_size_modifier)

# 如果设置过小（如0.10），征召出来的单位strength会太小，容易被秒杀
