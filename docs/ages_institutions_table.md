## 详细说明

### 1. 聚合时代 (Age of Aggregations) - 1585-1645

**时代特征**：这个时代见证了政治权力的地方化、全球贸易网络的建立以及防御工事技术的革新。

- **行政制度：省级官僚制 (Provincial Bureaucracy)**
  - 起源地：德里 (Delhi)
  - 特点：权力下放，地方行政机构的重要性增强

- **经济制度：全球贸易 (Global Trade)**
  - 起源地：万丹 (Bantam)
  - 特点：跨洲贸易网络的形成，商业中心的重要性提升

- **军事制度：意大利式防御工事 (Trace Italienne)**
  - 起源地：佛罗伦萨 (Florence)
  - 特点：星形要塞设计，防御工事技术的革命性改进

### 2. 危机时代 (Age of Crises) - 1645-1705

**时代特征**：这个时代充满了政治动荡、经济重组和军事变革，各国都在应对内外部危机。

- **行政制度：法律中央集权 (Legal Centralization)**
  - 起源地：北京 (Beijing)
  - 特点：加强中央政府对法律和行政的控制

- **经济制度：商队驿站制造业 (Caravanserai Manufacturing)**
  - 起源地：伊斯法罕 (Isfahan)
  - 特点：通过商队驿站网络促进制造业和贸易

- **军事制度：军事革命 (Military Revolution)**
  - 起源地：斯德哥尔摩 (Stockholm)
  - 特点：军事组织和战术的根本性变革

### 3. 扩张时代 (Age of Expansions) - 1705-1765

**时代特征**：这个时代以领土扩张、商业网络扩展和殖民活动为特征。

- **行政制度：财政军事国家 (Fiscal-Military State)**
  - 起源地：伦敦 (London)
  - 特点：国家财政与军事力量的紧密结合

- **经济制度：商业网络 (Mercantile Networks)**
  - 起源地：桑给巴尔 (Zanzibar)
  - 特点：广泛的商业网络和贸易联系

- **军事制度：设防定居点 (Fortified Settlements)**
  - 起源地：孟加拉 (Bengal)
  - 特点：在扩张地区建立防御性定居点

### 4. 革命时代 (Age of Revolutions) - 1765-1836

**时代特征**：这个时代见证了政治、经济和军事领域的革命性变化。

- **行政制度：宪政自由主义 (Constitutional Liberalism)**
  - 起源地：费城 (Philadelphia)
  - 特点：宪政制度和自由主义的兴起

- **经济制度：工业化 (Industrialization)**
  - 起源地：布莱克本 (Blackburn)
  - 特点：工业革命，制造业的机械化

- **军事制度：全民动员 (Levée en Masse)**
  - 起源地：巴黎 (Paris)
  - 特点：大规模征兵，全民参与战争

## 制度类型分布

### 按制度类型分类

**行政制度演进**：
1. 省级官僚制 (Delhi) → 2. 法律中央集权 (Beijing) → 3. 财政军事国家 (London) → 4. 宪政自由主义 (Philadelphia)

**经济制度演进**：
1. 全球贸易 (Bantam) → 2. 商队驿站制造业 (Isfahan) → 3. 商业网络 (Zanzibar) → 4. 工业化 (Blackburn)

**军事制度演进**：
1. 意大利式防御工事 (Florence) → 2. 军事革命 (Stockholm) → 3. 设防定居点 (Bengal) → 4. 全民动员 (Paris)
