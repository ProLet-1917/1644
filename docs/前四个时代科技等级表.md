# 前四个时代科技等级表

本文档列出前四个时代的所有科技，包括依赖关系和科技等级分类。

## 科技等级划分规则

- **等级1**：基础科技（depth=0），无依赖
- **等级2**：直接依赖基础科技（等级1）
- **等级3**：依赖等级2的科技
- **等级4**：依赖等级3的科技
- **等级5+**：更深层的依赖

## 总结

### 各时代基础科技数量
- **传统时代**：6个基础科技
- **文艺复兴时代**：4个基础科技
- **发现时代**：3个基础科技
- **宗教改革时代**：3个基础科技

### 科技等级分布
- **等级1**：基础科技（depth=0），共16个
- **等级2-4**：主要科技树分支
- **等级5+**：高级科技和特殊科技

### 使用建议
1. 开局科技等级1：研究所有基础科技
2. 开局科技等级2：研究基础科技 + 一级依赖科技
3. 开局科技等级3：研究到二级依赖科技
4. 开局科技等级4：研究到三级依赖科技
5. 更高等级：根据国家需求选择特定科技树
